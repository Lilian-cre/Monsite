Comment faire un site web ?

Guide simple pour créer votre premier site internet

Créer son premier site web

Découvrez les principales étapes pour concevoir un site en HTML.

1. Préparer les outils

Pour créer un site web, vous avez besoin d'un éditeur de code comme Visual Studio Code, Notepad++ ou même le Bloc-notes. Vous aurez également besoin d'un navigateur web pour tester votre site.

2. Créer un fichier HTML

Créez un fichier nommé index.html. Ce fichier représentera la page principale de votre site web. 

<!DOCTYPE html>
<html>
<head>
    <title>Mon site</title>
</head>
<body>
    <h1>Bienvenue sur mon site</h1>
</body>
</html>

3. Ajouter du contenu

Vous pouvez ajouter des titres, des paragraphes, des images, des liens et des listes afin de rendre votre site plus complet.

4. Personnaliser le design

Grâce au CSS, vous pouvez modifier les couleurs, les polices, les tailles et l'organisation des éléments pour obtenir un site plus agréable à consulter.

5. Tester le site

Enregistrez votre fichier puis ouvrez-le dans un navigateur (Chrome, Firefox, Edge...). Vérifiez que tous les éléments s'affichent correctement.
