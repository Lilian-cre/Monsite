Guide des balises HTML

Découvrez toutes les balises HTML expliquées simplement



<a>

Crée un lien hypertexte permettant de naviguer vers une autre page, une section du document, un fichier ou une adresse externe.

<abbr>

Représente une abréviation et peut contenir sa forme développée via un attribut explicatif.

<article>

Définit un contenu autonome pouvant être distribué ou compris indépendamment.

<aside>

Contient des informations secondaires ou complémentaires liées au contenu principal.

<audio>

Permet d’intégrer un contenu audio avec des contrôles de lecture dans une page web.

<a>

Crée un lien hypertexte permettant de naviguer vers une autre page, une section du document, un fichier ou une adresse externe.

<abbr>

Représente une abréviation et peut contenir sa forme développée via un attribut explicatif.

<article>

Définit un contenu autonome pouvant être distribué ou compris indépendamment (article, billet, post).

<aside>

Contient des informations secondaires ou complémentaires liées au contenu principal (encadrés, notes, publicités).

<audio>

Permet d’intégrer un contenu audio avec des contrôles de lecture dans une page web.

<b>

Met le texte en gras sans lui donner d’importance sémantique particulière.

<blockquote>

Indique une citation longue provenant d’une source externe.

<body>

Contient tout le contenu visible affiché dans le navigateur.

<br>

Insère un saut de ligne dans un texte sans créer de nouveau paragraphe.

<button>

Crée un bouton interactif déclenchant une action ou un événement.

<code>

Affiche un extrait de code informatique en police monospace.

<datalist>

Fournit une liste de suggestions pour un champ de saisie.

<dd>

Décrit la définition d’un terme dans une liste de définitions.

<details>

Crée un bloc repliable permettant d’afficher ou masquer du contenu.

<dialog>

Représente une fenêtre de dialogue ou une boîte modale interactive.

<div>

Conteneur générique sans signification sémantique utilisé pour structurer la mise en page.

<dl>

Liste de définitions regroupant des termes et leurs descriptions associées.

<dt>

Représente un terme à définir dans une liste de définitions.

<em>

Ajoute une emphase au texte, indiquant une importance ou une insistance.

<fieldset>

Regroupe plusieurs champs de formulaire liés logiquement.

<figcaption>

Fournit une légende descriptive pour une figure ou une illustration.

<figure>

Contient un contenu illustratif indépendant comme une image, un schéma ou un graphique.

<footer>

Définit le pied de page contenant des informations complémentaires ou des liens.

<form>

Permet de créer un formulaire pour collecter et envoyer des données utilisateur.

<h1> à <h6>

Représentent les titres hiérarchiques d’une page, du plus important au moins important.

<head>

Contient les métadonnées et ressources invisibles de la page.

<header>

Définit l’en-tête d’une page ou d’une section.

<hr>

Représente une séparation thématique entre deux parties de contenu.

<html>

Élément racine qui contient tout le document HTML.

<img>

Affiche une image avec possibilité de texte alternatif.

<input>

Champ interactif permettant à l’utilisateur de saisir des données.

<label>

Associe un texte explicatif à un champ de formulaire.

<legend>

Définit un titre pour un groupe de champs dans un formulaire.

<li>

Représente un élément individuel dans une liste.

<link>

Relie des ressources externes comme des fichiers CSS ou icônes.

<main>

Contient le contenu principal unique et central de la page.

<mark>

Surligne un texte pour attirer l’attention sur une information importante.

<meta>

Fournit des métadonnées sur la page (encodage, description, SEO).

<nav>

Contient les liens principaux de navigation du site.

<ol>

Crée une liste ordonnée avec numérotation automatique.

<p>

Définit un paragraphe de texte.

<pre>

Affiche un texte préformaté en conservant les espaces et retours à la ligne.

<progress>

Indique l’avancement d’une tâche en cours.

<q>

Représente une citation courte intégrée dans le texte.

<script>

Permet d’intégrer ou exécuter du code JavaScript.

<section>

Regroupe un contenu thématique cohérent avec un titre.

<select>

Crée une liste déroulante de choix.

<small>

Représente un texte secondaire ou moins important.

<span>

Conteneur en ligne utilisé pour styliser une partie de texte.

<strong>

Indique un texte fortement important avec emphase sémantique.

<style>

Permet d’écrire du code CSS directement dans le document.

<table>

Structure des données sous forme de lignes et colonnes.

<textarea>

Champ de saisie multiligne pour le texte.

<time>

Représente une date ou une heure de manière lisible par les machines.

<title>

Définit le titre de la page affiché dans l’onglet du navigateur.

<ul>

Crée une liste non ordonnée avec des puces.

<video>

Intègre une vidéo avec contrôles de lecture dans la page.

<wbr>

Indique un point de rupture possible dans un mot long.
